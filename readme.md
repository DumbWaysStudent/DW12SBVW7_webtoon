# Sanstoon API

## Installation

Make sure you have installed Node.js on your machine. 

### Install modules
---
Run the code below.
```
$ npm install
```
or, if you using yarn
```
$ yarn install
```

### Create .env file
---
Create .env file in root folder and fill data below with you own.
```
PORT=your port
KEY=your secret key
```

## Run
Run code below to start server
```
$ node app.js
```
